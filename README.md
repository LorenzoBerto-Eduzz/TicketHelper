# Como instalar

1. Vá em Releases e baixe a versão desejada, clicando em `TicketHelper.zip`.
2. Mova o conteúdo (pasta com mesmo nome) a um local de preferência.
3. Em `chrome://extensions/`, ative o Modo de Desenvolvedor, clique em **Carregar sem compactação**, escolhendo a pasta salva.
4. Edite os atalhos na página de configuração como preferir.
5. Recarregue as abas de ticket/chat em uso.
